BUAT DESAIN UI PORTAL MAHASISWA UNIKOM (FIGMA / GOOGLE STITCH)
===============================================================

【DESKRIPSI APLIKASI】
Aplikasi portal mahasiswa berbasis web dengan tema modern, bersih, dan profesional. Target pengguna adalah mahasiswa aktif. Warna dominan biru (#1D4ED8) dengan aksen amber (#F59E0B). Font utama: Plus Jakarta Sans.

【COLOR PALETTE】
- Primary: #1D4ED8 (Blue-700)
- Primary Dark: #1E3A8A (Blue-900)
- Accent: #F59E0B (Amber-500)
- Success: #10B981 (Emerald-500)
- Danger: #EF4444 (Red-500)
- Text Primary: #374151 (Gray-700)
- Text Secondary: #6B7280 (Gray-500)
- Text Muted: #9CA3AF (Gray-400)
- BG Card: #FFFFFF
- BG Surface: #F9FAFB (Gray-50)
- Border: #E5E7EB (Gray-200)
- Dark mode: bg-gray-900, card bg-gray-800, text putih

【TYPOGRAPHY】
- Font: Plus Jakarta Sans, sans-serif
- Monospace: JetBrains Mono (untuk NIM/kode)
- Skala: 10px (badge), 12px (label), 14px (body), 18px (subtitle), 24px (title), 36px (hero)

【SPACING & RADIUS】
- Spacing: 4px (xs), 8px (sm), 16px (md), 24px (lg), 32px (xl)
- Radius: 8px (sm), 12px (md), 16px (lg), 24px (xl), 9999px (full)
- Shadow card: 0 1px 3px rgba(0,0,0,0.06)
- Shadow elevated: 0 4px 12px rgba(0,0,0,0.08)

- Hamburger icon di Navbar (kiri atas)

【HALAMAN & KOMPONEN】

=== 1. LOGIN PAGE ===
Layout: Full screen, centered, dua kolom (hero image kiri, form kanan)
- Kiri: Background gradient biru, ilustrasi/ikon dekoratif besar, teks branding "Portal Mahasiswa UNIKOM"
- Kanan: Card putih, logo + judul "Masuk Portal", form NIM + Password (dengan toggle show/hide), tombol "Masuk" (bg-primary, rounded-xl, py-2.5), link "Lupa Password?", loading spinner
- Warna background kanan: bg-gray-50

=== 2. DASHBOARD ===
- Greeting banner: Gradient bg dinamis (pagi/siang/sore/malam), teks sambutan + nama, emoji dekoratif besar
- 4 Stat Cards (grid 4 kolom): SKS Ditempuh, IPK Kumulatif, Semester Aktif, Kehadiran Rata-rata. Masing-masing: icon lingkaran dengan bg warna, nilai besar (text-2xl/3xl), label small
- Warning banner KRS (bg-amber-50, border-amber-200, dismissible)
- Aktivitas Akademik Penting: Grid 2-3 kolom, card dengan left border accent color, icon + judul + deskripsi
- Persetujuan Dosen Wali: Card hijau (bg-emerald-50), icon centang, status sukses
- Akses Cepat: 7 tombol aksi cepat dengan icon + label (bulat/rounded-full)
- Bagian Favorit: Grid item yang di-bookmark (bintang aktif)
- Berdasarkan Aktivitasku: Rekomendasi berdasarkan frekuensi klik
- Jadwal Kuliah Hari Ini: Timeline vertikal, card per mata kuliah dengan jam + nama + ruang + dosen
- Riwayat Aktivitas Terbaru: List feed dengan icon + waktu + deskripsi

=== 3. AKADEMIK (5 tabs) ===
Layout: Tab bar horizontal dengan animated indicator (framer-motion layoutId)

Tab - KRS:
- Status summary: 4 card grid (Status Pembayaran, Batas SKS, Persetujuan, Periode)
- Stepper 4 langkah horizontal (lingkaran bernomor, garis penghubung)
- Tabel checklist: No, Kode MK, Nama Mata Kuliah, SKS, Dosen, Hari/Jam, Ruang, checkbox
- Tombol: Simpan (primary), Unduh PDF (secondary)

Tab - Jadwal Kuliah:
- Timetable grid: 6 kolom (Senin-Sabtu) x 11 baris jam (07:00-17:00)
- Cell berisi nama MK (text kecil), warna latar berbeda per MK
- Legend warna di bawah

Tab - Nilai:
- Dropdown pilih semester
- 4 Stat Cards: IP Semester, Total SKS, IP Kumulatif, Total SKS Lulus
- Tabel: Kode, Nama MK, SKS, Nilai Huruf (warna: A=hijau, B=biru, C=kuning, D=oranye, E=merah), Nilai Angka, Bobot

Tab - Absensi:
- Info banner: "Minimal kehadiran 75%"
- Tabel: Mata Kuliah, Total Pertemuan, Hadir, Izin, Alpha, Persentase, Status (Aman=hijau, Perhatian=kuning, Bahaya=merah)

Tab - Pengisian Kelas:
- Per Mata Kuliah: nama MK + SKS, toggle button pilih kelas (Kelas A / B / C)
- Tombol Simpan

=== 4. KEUANGAN (2 tabs) ===
Tab - Tagihan:
- Active bill card: Background gradient red, nominal besar, jatuh tempo, tombol "Bayar Sekarang"
- Ringkasan Pembayaran: Total tagihan, total dibayar, sisa

Tab - Riwayat Pembayaran:
- Filter: Semua / Lunas / Belum Lunas (pill buttons)
- Tabel: Tanggal, Keterangan, Jumlah, Status (lunas=hijau, belum=merah), Aksi (Unduh link)

=== 5. LAYANAN (5 tabs) ===
Tab - Layanan:
- 6 Service Cards: Icon emoji besar, judul, deskripsi singkat, link "Ajukan"
- Modal form saat klik Ajukan: Icon, judul, input Tujuan Penggunaan, textarea Catatan, tombol Kirim
- Riwayat pengajuan: tabel status permohonan

Tab - Kuesioner:
- List survey dengan status pill (Belum Diisi / Selesai), tombol "Isi" untuk yang pending

Tab - Syarat Wisuda:
- Checklist persyaratan dengan emoji (✅/❌) dan status

Tab - FAQ:
- 2 column grid accordion: Pertanyaan (clickable), jawaban expand dengan animasi
- Help card: Icon + teks bantuan + tombol guide

Tab - Chatbot:
- Chat panel: Header dengan nama bot + icon, area chat (bubble user kanan biru, bot kiri putih), typing indicator (3 dots animation), input field + send button
- Quick prompts: Pill buttons di atas input
- Handoff form: Nama, Email, Pesan (muncul saat bot tidak bisa jawab)

=== 6. PROFIL (3 tabs) ===
Tab - Data Diri:
- Avatar lingkaran (initial name), nama + NIM
- Grid info fields: NIM, Nama, Tempat Lahir, Tanggal Lahir, Jenis Kelamin, Agama, Alamat, No HP, Email, Angkatan, Status
- Tombol "Perbarui Data" (secondary)

Tab - Keamanan:
- Form ganti password: Password Lama, Password Baru, Konfirmasi Password
- Toggle 2FA (switch component)

Tab - Tampilan:
- Dark Mode toggle (switch)
- Ukuran Font: Kecil / Normal / Besar (pill radio buttons)
- Atur Tampilan Dashboard: 4 toggle switches (sembunyikan card tertentu)
- Frekuensi Data Aktivitas: Bar chart visual
- Bahasa: Indonesia / English (pill radio)

=== 7. NOTIFICATIONS ===
Panel dropdown dari Navbar: List notifikasi dengan icon (kuning/merah/hijau/biru sesuai tipe), waktu relatif, tombol dismiss

=== 8. AI CHAT WIDGET ===
Floating action button (fixed bottom-right, circle 56px, bg-primary)
Panel chat: Sama seperti chatbot di Layanan, bisa diakses dari mana saja
