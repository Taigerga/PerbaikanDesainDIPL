1. PENDAHULUAN
--------------------------------------------------------------------------------
Proyek ini adalah prototype redesign dari portal akademik "My Academic UNIKOM"
(my.unikom.ac.id) sebagai tugas UTS mata kuliah Desain Interaksi Perangkat
Lunak - Universitas Komputer Indonesia (UNIKOM).

Mahasiswa : Muhammad Rizki (NIM: 10123030)
Teknologi : React 19, Vite 8, Tailwind CSS v4, Framer Motion, Lucide React
Jenis     : Single-Page Application (SPA) frontend-only (pure UI prototype)
Data      : Hardcoded dummy data (tanpa backend/database)
Deploy    : GitHub Pages


2. STRUKTUR HALAMAN (7 Halaman)
--------------------------------------------------------------------------------
Rute (HashRouter) :
  #/           -> Login
  #/dashboard  -> Dashboard
  #/akademik   -> Akademik (5 tab)
  #/keuangan   -> Keuangan (2 tab)
  #/layanan    -> Layanan (5 tab)
  #/profil     -> Profil (3 tab)
  #/bantuan    -> Bantuan (redirect ke Layanan > FAQ)

4. KOMPONEN UI REUSABLE
--------------------------------------------------------------------------------
  - Sidebar          : Navigasi kiri dengan grup collapsible + animasi
  - Navbar           : Header sticky (search global, notifikasi, profile, dark mode)
  - BottomNav        : Navigasi bawah mobile (5 item)
  - Breadcrumb       : Navigasi breadcrumb dinamis (semua halaman)
  - DashboardCard    : Kartu stat gradient dengan icon
  - AIChatWidget     : Floating chatbot FAB + panel slide-up
  - NotificationPanel: Dropdown notifikasi dengan prioritas + mark-as-read
  - OnboardingTour   : Modal 4 langkah (muncul otomatis setelah login)
  - ConfirmDialog    : Modal konfirmasi (summary items, variant danger)
  - SuccessPage      : Halaman sukses dengan centang + ringkasan + next steps
  - ErrorMessage     : Tampilan error dengan penyebab + solusi
  - Tooltip          : Tooltip info (4 posisi)
  - Skeleton         : Loading placeholder (pulse animation)
  - PageSkeleton     : Loading placeholder untuk halaman penuh


5. DESAIN SISTEM (Design Tokens)
--------------------------------------------------------------------------------
  Warna:
    - Primary   : #1D4ED8 (biru)
    - Accent    : #F59E0B (kuning/emas)
    - Success   : #10B981 (hijau)
    - Danger    : #EF4444 (merah)
    - Background: #F8FAFC (terang) / #0F172A (gelap)

  Tipografi:
    - UI     : Plus Jakarta Sans (sans-serif)
    - Kode   : JetBrains Mono (monospace, untuk NIM)

  Spacing: xs(4px) - sm(8px) - md(16px) - lg(24px) - xl(32px) - 2xl(40px)

  Border Radius: sm(8px) - md(12px) - lg(16px) - xl(24px) - full(9999px)

  Shadow: card, elevated, modal


6. FITUR INTERAKTIF & LINTAS HALAMAN
--------------------------------------------------------------------------------
  - Dark Mode             : localStorage-persisted, transisi smooth CSS
  - Pencarian Global      : Synonym-based search dengan context enrichment
                            (mendeteksi "ipk" -> tampilkan IPK, "bayar" -> tagihan)
  - Keyboard Shortcuts    : Shift+G (Dashboard), Shift+A (Akademik),
                            Shift+K (Keuangan), Shift+L (Layanan),
                            Shift+P (Profil), Shift+B (Bantuan),
                            / (focus search), Escape (blur)
  - Animasi Halaman       : Framer Motion AnimatePresence (fade + slide-up)
  - Toast Notifications   : Stackable, auto-dismiss (success/warning/error/info)
  - Bookmark / Favorit    : Star toggle pada quick access (localStorage)
  - Pelacakan Frekuensi   : LocalStorage-based usage tracking untuk rekomendasi
  - Font Size Customization: 3 level via CSS data attribute
  - Visibilitas Dashboard : 4 toggle untuk menyembunyikan section dashboard
  - Loading State         : Skeleton pulse pada setiap halaman
  - Empty State           : Tampilan khusus saat data kosong
  - Error State           : Tampilan error dengan penyebab + solusi


7. RESPONSIVE DESIGN
--------------------------------------------------------------------------------
  Desktop : Sidebar penuh + Navbar + konten utama
  Tablet  : Sidebar icon-only + Navbar + konten utama
  Mobile  : BottomNav + drawer menu + konten penuh

  Breakpoint: Tailwind default (sm: 640px, md: 768px, lg: 1024px, xl: 1280px)


8. AKSESIBILITAS (WCAG 2.1 AA)
--------------------------------------------------------------------------------
  - Kontras warna memadai (text/bg)
  - Fokus visible pada elemen interaktif
  - Semantic HTML structure
  - Label pada form inputs
  - Status dan error messages yang jelas
  - Font size adjustable
